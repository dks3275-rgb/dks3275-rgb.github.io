1. 삭제 요청 방법

사용자는 다음 두 가지 방법 중 한 가지를 선택하여 계정 및 데이터의 삭제를 요청할 수 있습니다.

A방법 1: 앱 내 1:1 상담 기능 이용 (권장)

  1. 앱 메인 화면에서 메뉴 → "1:1 상담 신청하기" 클릭
  2. 제목: "계정 삭제 요청" 입력
  3. 본문에 삭제할 정보(예: 이름, 학과)를 적고 신청
  4. 관리자가 24~72시간 이내 확인 후 삭제 처리

B방법 2: 이메일로 직접 요청

아래 이메일로 다음 정보를 보내주세요.

  • 제목: [AU 일학습병행] 계정 삭제 요청
  • 본문 포함 내용: 가입 시 입력한 이름, 학과(또는 카테고리), 가입 시점(대략)
📧 이메일 보내기 (dks3275@gmail.com)

C방법 3: 앱 직접 삭제 (즉시 효력)

기기에서 앱을 삭제하면 다음 데이터는 즉시 자동 삭제됩니다.

  • 로컬 가입 정보 캐시 (localStorage)
  • OneSignal 푸시 구독 (자동 비활성화)
  • 기기 식별자 정보

단, 서버(Firebase)에 저장된 정보는 위 방법 1 또는 2를 통해 별도 요청이 필요합니다.

2. 삭제되는 데이터 유형

데이터 항목삭제 시점
이름, 학과(학년), 카테고리요청 즉시
연락처(예비신입생/전담인력)요청 즉시
OneSignal 푸시 구독 ID요청 즉시
기기 식별자(deviceId)요청 즉시
1:1 상담 내역상담 종료 후 1년 (요청 시 즉시)
외부평가 모의시험 응시 기록응시 후 5년 (교육 운영 보존 의무)*
* 외부평가 응시 기록은 일학습병행 공동훈련센터 운영 규정에 따라 5년간 보존이 의무화되어 있어, 사용자 요청에도 즉시 삭제되지 않을 수 있습니다. 보존 기간 종료 시 자동 파기됩니다.

3. 보관되는 데이터 (선택적 보관)

삭제 요청 시 다음 정보는 법령상 의무 또는 분쟁 대비를 위해 일정 기간 보관될 수 있습니다.

유형보관 기간법적 근거
외부평가 응시 기록5년일학습병행 운영 규정
분쟁 처리 관련 기록3년전자상거래법 (해당 시)

위 항목 외에는 모두 즉시 영구 삭제됩니다.

4. 처리 기간

  • 요청 접수 후 최대 72시간 이내 처리
  • 처리 완료 시 동일 채널(상담 또는 이메일)로 회신

5. 데이터 일부 삭제 요청

전체 계정 삭제가 아닌 특정 정보(예: 연락처만 삭제, 학과 정보 변경)를 요청할 경우, 위 방법 1 또는 2의 절차로 동일하게 요청해주시면 됩니다.

또한 앱 메인 화면의 "수정" 버튼을 통해 직접 본인 정보를 변경할 수도 있습니다.
