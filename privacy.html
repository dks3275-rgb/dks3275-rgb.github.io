AU 일학습병행 공동훈련센터 앱
본 개인정보 처리방침은 안산대학교 일학습병행 공동훈련센터에서 운영하는 PWA/모바일 앱 ("AU 일학습병행", 이하 "앱")에 적용됩니다.
최종 개정일: 2026년 5월 1일

1. 수집하는 개인정보 항목

앱은 서비스 제공을 위해 다음 정보를 수집합니다.

대상수집 항목
재학생이름, 학과(학년)
졸업생이름, 졸업 학과, 졸업 년도, 현재 직장
예비신입생이름, 연락처, 현재 학교명, 관심 학과
학습기업 전담인력이름, 회사명, 직책, 담당 학생 이름
공통기기 식별자(deviceId, OneSignal ID), 알림 수신 동의 여부, 접속 일시

1:1 상담 신청·문의 등록 시 신청자가 작성한 상담 내용 및 연락처(선택)도 함께 저장됩니다.

2. 개인정보 수집 방법

  • 앱 가입 시 사용자가 직접 입력
  • 알림 수신 동의 시 OneSignal 자동 발급 식별자(onesignalId, deviceId)
  • 상담 신청·댓글 작성 시 사용자가 입력한 정보

3. 개인정보 수집 및 이용 목적

  1. 회원 식별 및 카테고리별 콘텐츠 제공 — 재학생/졸업생/예비신입생/전담인력별 맞춤 공지·자료실·메뉴 제공
  2. 알림 발송 — 학사 일정·공지사항·시험 안내·상담 답변 등 푸시 알림
  3. 1:1 상담 운영 — 신청자와 관리자 간 양방향 메시지 교환 및 상담 이력 관리
  4. 외부평가 모의시험 응시 기록 관리
  5. 훈련 만족도 조사 — 익명 또는 학과별 통계 분석

4. 개인정보 보유 및 이용 기간

회원 가입 시점부터 다음 시점까지 보유합니다.

  • 회원 탈퇴(앱 삭제) 시까지 — 회원 정보
  • 상담 종료 후 1년 — 상담·문의 내역
  • 외부평가 응시 후 5년 — 시험 응시 기록 (교육 운영 보존 의무)

관리자에게 직접 삭제 요청 시 즉시 파기됩니다 (1:1 상담 기능 또는 메일 문의).

5. 개인정보 제3자 제공 및 처리위탁

앱은 사용자의 개인정보를 외부에 제공하거나 판매하지 않습니다. 다만, 서비스 제공을 위해 다음 외부 서비스를 이용하며, 필수 정보가 해당 서비스에 저장됩니다.

업체역할저장 항목
Google Firebase (Firestore)회원 정보·상담·공지·자료 데이터베이스1번 항목의 전체 정보
OneSignal푸시 알림 발송OneSignal ID, 카테고리·이름·학년 태그
Cloudinary공지사항 첨부 이미지 호스팅관리자 업로드 이미지
Cloudflare WorkersOneSignal API 호출 프록시발송 메시지 (영구 저장 X)
GitHub Pages앱 호스팅 및 자료실 파일 저장관리자가 등록한 학습 자료 파일

각 업체는 자체 개인정보 정책에 따라 정보를 보호합니다. 자세한 내용은 각 업체의 개인정보 처리방침을 참고해주세요.

6. 개인정보의 파기 절차 및 방법

  • 전자 정보: Firestore 및 OneSignal에서 즉시 영구 삭제 (deleteDoc / removeUser)
  • 디바이스 측 정보(localStorage): 앱 삭제 시 자동 삭제

7. 사용자의 권리 및 행사 방법

사용자는 언제든지 다음 권리를 행사할 수 있습니다.

  • 본인 정보 열람·수정 — 앱 메인 화면의 "수정" 버튼
  • 알림 수신 거부 — 앱 내 "알림 끄기" 또는 OS 설정
  • 회원 탈퇴 및 삭제 요청 — 1:1 상담 기능 또는 아래 연락처로 요청

8. 개인정보 보호 조치

  • HTTPS 암호화 통신
  • Firebase 자체 보안 인프라 사용
  • 관리자 권한은 OneSignal 태그 기반 인증으로 제한
  • 최소 필요 정보만 수집

9. 만 14세 미만 아동의 개인정보

본 앱은 안산대학교 일학습병행 학생(원칙적 만 18세 이상)을 대상으로 합니다. 만 14세 미만 아동의 가입은 받지 않으며, 오인 가입이 확인되면 즉시 삭제 처리합니다.

10. 쿠키 및 유사 기술

앱은 사용자 식별 및 알림 수신을 위해 다음 클라이언트 측 저장 기술을 이용합니다.

  • localStorage — 가입 정보 캐시 (다음 앱 진입 시 자동 인식)
  • OneSignal IndexedDB — 푸시 구독 상태
  • Service Worker — OneSignal 알림 처리

모두 사용자 기기 내부에만 저장되며, 외부 광고·트래킹 목적으로는 사용되지 않습니다.

11. 개인정보 처리방침 변경

본 처리방침은 법령·서비스 변경에 따라 수정될 수 있으며, 변경 시 앱 공지사항을 통해 안내합니다.
